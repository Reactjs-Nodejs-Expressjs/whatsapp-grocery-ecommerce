# Groceria WA

Frontend ecommerce app with WhatsApp-first checkout and admin order tracking.

## Quick Start

### 1) Install

```bash
npm install
```

### 2) Start Live Local Server

```bash
npm run dev
```

Open URL shown in terminal (usually `http://localhost:5173`).

### 3) Build

```bash
npm run build
```

### 4) Preview Build

```bash
npm run preview
```

### 5) Lint

```bash
npm run lint
```
